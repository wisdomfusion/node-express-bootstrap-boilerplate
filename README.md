# node-express-bootstrap-boilerplate

Node.js/Express.js MVC skeleton project with front end assets built with Bootstrap 4, webpack 5 and jQuery.

## app_server

Copy `.env.example` to new `.env` file.

```sh
npm i
npm i -g nodemon
nodemon ./bin/www
```

DEBUG
```sh
$env:DEBUG='*'; nodemon .\bin\www
```

## app_assets

```
cd app_assets/
npm i
npm run watch
```
